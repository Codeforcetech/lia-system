# 実ユーザー確認用チェックリスト（質問リスト）

目的：Liaが「次に送るLINEを考える相棒」になっているか確認する。

## 1. 生成文面の品質

- 生成されたLINE文面は **自然**でしたか？
- 営業っぽさ（押し売り感）は **強すぎません**でしたか？
- 距離感（丁寧すぎ/馴れ馴れしすぎ）は **ちょうど良い**ですか？
- 1案あたりの長さ（1〜3文）は **送りやすい**ですか？
- 「そのまま送る」なら、どの案を選びますか？理由は？
- 絵文字量（文体） は **ちょうど良い**ですか？
- 不快/過激/危ない表現はありませんでしたか？
- 直近メモの反映は **自然**でしたか？（不自然に詰め込まれていないか）

## 2. 毎日使うイメージ

- ホームの「今日の営業候補」は **役に立ちそう**ですか？

- 逆に「見ないかも」と感じる理由は？
- 1日に何回くらい開きそうですか？

## 3. 面倒なところ / 迷うところ

- 顧客登録で面倒だった入力は？
- メモ追加・来店追加は **手間**に感じましたか？
- 生成画面で迷った項目（目的/文体/追加指示）は？
- 「次に何をすればいいか」が分かりにくい場面は？

## 4. もっと欲しい機能（優先度つき）

以下から「欲しい順」に3つ選んでください。

- 既読/返信状況のメモ（返信薄めなどの管理）
- 生成履歴の検索（目的/文体/期間）
- 生成文面の編集して保存（テンプレ化）
- 毎日のリマインド（今日連絡すべき一覧を固定）
- 誕生日リマインドの精度UP
- LINE連携（自動下書き/送信）
- 複数ユーザー（キャストごと）
